Selecting cells:
  • Click/Drag: select cells
  • Ctrl+Click/Drag: add/remove from selection
  • Ctrl+I: invert selection
  • Arrows / Tab / Shift+Tab: move single selected cell
Entering numbers:
  • 1-9: Set selection to number
  • Ctrl+{1-9}: Add/remove number from pencil marks
  • Delete/Backspace: Mark cell as being any of 1-9
  • Ctrl+Delete/Backspace: Remove all possibilities from cell
Other:
  • Ctrl+Z/Y: Undo/Redo
Rule definitions:
  • Antiknight: Same digits cannot be a chess knight move apart
  • Antiking: Same digits cannot be a chess king move apart
  • Diagonals: The two diagonals contain 1-9
  • Nonconsecutive: Orthogonally connected squares cannot contain consecutive digits
  • Digits not in same position: Same digits cannot appear in the same position in different blocks
Strategy definitions:
  • Obvious: Eliminate possibilities that fall directly out of the rules
  • Intersections (pointing, box/line reduction): When a digit is restricted to some set of positions (e.g. one row in a block), eliminate positions that are ruled out regardless of where the digit is in the group (e.g. eliminate rest of row).
  • Naked sets: If N cells contain only N possibilities between them, the other cells can't contain those N possibilities.
  • Fish (X-Wing, Swordfish, Jellyfish): Suppose in N rows, a digit can only appear in the same N columns. Then in those N columns, other rows are not possible.
  • Hidden singles: For each digit, see if it can only go in one place for some group
